# API REST

## gestion de la base de datos

al iniciar el codigo deberemos migrar las bases ``php artisan migrate`` para que la base de datos ``chollos`` se llene de las tablas que se presentan en el proyecto.

Ingresaremos el ``.sql`` que deje en la tarea para así poder trababar sobre los datos de forma correcta

## Enlace para ver el api rest

### url para ver la informacion

ingresaremos a la url los siguientes parametros para ver la informacion de la api

- /api/chollos
- /api/categorias
