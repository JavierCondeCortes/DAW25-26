# Crear una tienda

hay que crear una tienda que conste con una pantalla de login y register.
al iniciar sesion tenemos 3 opciones, ver los productos, agregar y eliminar.
en la opcion de ver los productos tiene que aparecer el nombre, descripcion, cantidad.

al hacer click en los productos nos muestra una web donde el producto se mostrará en una web diferente.
en esa web podremos agregar al carrito y poner una reseña.

el login debe permanecerse con una session.
una cookie con el ultimo producto visitado.
